# ml_policejob
 police job made for redem_roleplay



# Installation
1. Clone this repository.
2. Extract the zip.
3. Put ml_policejob to your resource folder.
4. Add "start ml_policejob" in your "server.cfg".
5. Open menu with [U]
6. Open menu at pd with [G]

# Required resource
- redem_roleplay
